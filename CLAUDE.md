# CLAUDE.md - Project Instructions

## Git Commit Convention

เมื่อทำงานเสร็จแต่ละ task ให้ commit ด้วย pattern:

```
[{Task ID}] {Task Name}
```

**Examples:**
- `[P1-01] Create Monorepo Directory Structure`
- `[P1-02] Initialize Go Service Modules`
- `[P2-01] Redis Lua Script - Reserve Seats`

**Rules:**
- Commit ทันทีเมื่อทำ task เสร็จ
- ไม่ต้องใส่ Claude signature หรือ co-author
- ถ้า task มีหลาย sub-tasks ให้ commit รวมเมื่อทำ task หลักเสร็จ
